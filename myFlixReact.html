myFlix App (React)

Description

  • The goal of this project was to build the client-side for a web application focused on movies.
  • This client app was supposed to use the previously built myFlix API.
  • It is an SPA (single-page application) that provides a user with login/signup view, main view with a list of available movies fetched from the database, and profile view where the user can view and alter their account information, and list of their favourite movies which they can also remove and further add to the list.
  • Each movie also shows detailed information about itself upon clicking on its card.
  • Along with the technologies used to built the API, this is a representation of the MERN stack utilization.

Key Features

  • Returns ALL movies to the user (each movie item with an image, title, and description).
  • Filtering the list of movies with a “search” feature.
  • Ability to select a movie for more details.
  • Ability to log out.
  • Ability to navigate to Profile view.
  • Returns data (description, genre, director, image) about a single movie to the user.
  • Allows users to add a movie to their list of favorites.
  • Allows users to log in with a username and password.
  • Allows new users to register (username, password, email, date of birth).
  • Displays user registration details.
  • Allows users to update their info (username, password, email, date of birth).
  • Displays favorite movies.
  • Allows users to remove a movie from their list of favorites.
  • Allows existing users to deregister.

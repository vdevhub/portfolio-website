Pokédex

Description

  • This Pokédex App is a simple collection of Pokémons.
  • Each Pokémon is placed on its own card which displays the Pokémon’s basic characteristics in a modal upon clicking.
  • All details are fetched from an external API.
  • Bootstrap is used for styling.
Project on GitHub
Project Live

Key Features

  • Load data from an external source (API).
  • View a list of items.
  • On user action (e.g., by clicking on a list item), view details for that item.
